A continuación se explica cómo se crean máquinas y sincronizan con el programa.

Crear máquinas:

1. Ejecutar MaquinaBerkeley.java
2. Asignar Dirección
3. Asignar Puerto

Repetir este proceso tantas veces como máquinas se desee crear.

Sincronizar máquinas:

1. Ejecutar SincronizadorBerkeley.java
2. Agregar máquinas con sus respectivas direcciones y puertos
3. Elegir la opción de Sincronizar máquinas
